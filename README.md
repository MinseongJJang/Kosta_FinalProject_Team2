
# Kosta_FinalProject_Team2(Academy Helper)
- 개요
```
IT관련 교육을 희망하는 취업준비생들의 교육기관 및 교육과정 선정을 돕기위한 교육기관 비교 플랫폼   
```
- 주제 선정 이유
```
국내 많은 IT 교육 기관들과 수많은 교육이 있습니다.  
IT 취업을 희망하는 모든 사람들은  
짧지 않은 기간의 교육을 위해 IT 교육 과정과 교육 기관 선택에 신중합니다.  
하지만,  
해당 교육과정이 스스로에게 적합한 교육이며,  
생각한 기관의 환경이 생각과 일치하는지 모릅니다.  
모든 사람들(취업준비생)이 수많은 IT기관, IT교육 중 하나를 선택해야 하지만  
선택을 위한 투명하고 솔직한 자료가 많이 부족하다.  
```

# 프로젝트 팀원 및 역할
```
PM : 강민성(프로젝트 총괄)    
PL : 이성열(프로젝트 기술관련)  
DM : 윤준상(문서 및 자료)    
DL : 신현수(문서 및 자료)     
GL : 김성민(형상관리)     
CL : 이정빈(디자인)  
```
